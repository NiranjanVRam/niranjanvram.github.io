Emulated CAPWAP System

The Emulated CAPWAP System is a software-based simulation of the CAPWAP (Control and Provisioning of Wireless Access Points) protocol, designed for research, development, and testing purposes. It emulates both Access Controller (AC) and Wireless Termination Point (WTP) functionalities, supporting essential pre-defined messages such as heartbeat and discovery, along with full state transition handling. Capable of simulating up to 1,000 clients simultaneously, the system provides a scalable and reliable environment for exploring CAPWAP behavior without requiring physical hardware. Built with lightweight message handling and configuration support, it is suitable for academic research, developers, and network engineers alike. Fully functional and available as open-source code on GitHub, the system can be used in testing environments or as a proof-of-concept. While there are currently no planned updates, future improvements and additional features may be introduced.
